# 项目概览: word_select_dashboard/web-react

## 作用用途

`word_select_dashboard/web-react` 是单词清洗和运营管理的 React 后台页面，主要服务于管理、筛选、生成和查看任务结果。

## 主要功能

- 单词库、单词清洗和句子生成相关页面。
- AI 配置管理，支持选择和维护不同模型 provider。
- 查看执行任务、工作流记录、句子生成历史和任务状态。
- 管理用户、错词、已掌握单词、训练结果和完形结果。
- 触发句子生成、句子评分和 TTS 相关后台任务。

## 适合什么时候读

需要理解后台页面有哪些模块、运营管理入口或 React dashboard 与 Go API 的关系时读本文件。
