# 项目概览: rob_english_word_front

## 作用用途

`rob_english_word_front` 是英语抢词大项目的 Vue 主前端，面向普通用户提供登录、学习、游戏和学习记录页面。

## 主要功能

- 登录、注册和前端鉴权状态维护。
- 首页、抢词游戏、答题记录、训练结果等核心用户页面。
- 错词列表、已掌握单词列表和答题详情展示。
- 与 Java 后端 REST API 和 WebSocket 服务交互。
- 承载用户侧英语抢词和单词学习的主要体验。

## 适合什么时候读

需要理解用户主站页面范围、Vue 前端职责、用户侧功能入口或前后端页面链路时读本文件。
